# Philosophical Interview Framework

소크라틱 대화법을 활용한 철학적 인터뷰를 진행하고 블로그 포스트로 변환합니다.

## When to Activate

- User wants philosophical exploration / 철학적 탐구를 원할 때
- User says "소크라틱 대화" / "Socratic dialogue"
- User requests deep dive into a concept / 개념 깊이 탐구 요청
- User wants to explore artistic/philosophical ideas
- User says "이 주제로 인터뷰해줘" / "interview me about this"

## Socratic Question Framework

### 9 Question Types (9가지 질문 유형)

| # | Type | Korean | Example |
|---|------|--------|---------|
| 1 | Clarification | 명확화 | "What do you mean by...?" / "...가 무슨 뜻인가요?" |
| 2 | Assumptions | 가정 탐색 | "What are you assuming?" / "어떤 가정을 하고 계신가요?" |
| 3 | Evidence | 근거 | "What evidence supports this?" / "이를 뒷받침하는 근거는?" |
| 4 | Perspectives | 관점 | "How might others view this?" / "다른 사람들은 어떻게 볼까요?" |
| 5 | Implications | 함의 | "What follows from this?" / "이로부터 무엇이 따라오나요?" |
| 6 | About the Question | 질문 자체 | "Why is this question important?" / "이 질문이 왜 중요한가요?" |
| 7 | Origin | 기원 | "What led you to think this?" / "이렇게 생각하게 된 계기는?" |
| 8 | Examples | 예시 | "Can you give an example?" / "예를 들어주실 수 있나요?" |
| 9 | Synthesis | 종합 | "How does this connect to...?" / "이것이 ...와 어떻게 연결되나요?" |

## Interview Flow

### Phase 1: Opening (시작) - 5-10 minutes

**Goal:** Establish topic and context

**Techniques:**
- Ask opening question to understand the topic
- Use clarification questions (Type 1)
- Identify the core issue or interest
- Create safe space for exploration

**Example Openers:**
- "Tell me about this idea that's been on your mind."
- "이 생각에 대해 더 자세히 말씀해 주세요."
- "What draws you to this topic?"
- "이 주제에 끌리는 이유가 무엇인가요?"

### Phase 2: Exploration (탐구) - 15-30 minutes

**Goal:** Deep dive with varied question types

**Techniques:**
- Challenge assumptions (Type 2)
- Seek evidence (Type 3)
- Explore different perspectives (Type 4)
- Examine implications (Type 5)
- Ask about origins (Type 7)
- Request examples (Type 8)

**Rhythm:**
- 2-3 questions per type
- Follow interesting threads
- Allow natural tangents
- Return to core theme

**Deepening Prompts:**
- "Why do you think that matters?" / "왜 그게 중요하다고 생각하세요?"
- "What if the opposite were true?" / "반대가 사실이라면 어떨까요?"
- "What are you not saying?" / "말하지 않고 있는 것은 무엇인가요?"

### Phase 3: Synthesis (종합) - 5-10 minutes

**Goal:** Draw connections and insights

**Techniques:**
- Connect ideas discussed (Type 9)
- Identify key realizations
- Formulate conclusions or new questions
- Acknowledge what remains unresolved

**Closing Questions:**
- "What's the most important insight from our discussion?"
- "이 대화에서 가장 중요한 깨달음은 무엇인가요?"
- "What question remains unanswered?"
- "아직 답하지 못한 질문은 무엇인가요?"
- "How has your thinking changed?"
- "생각이 어떻게 바뀌었나요?"

## Philosophical Topics Library

### Epistemology (지식론)
- What can we truly know? / 우리가 진정으로 알 수 있는 것은?
- How do we know if something is true? / 무언가가 참인지 어떻게 알 수 있나요?
- What is the nature of belief? / 믿음의 본질은 무엇인가요?

### Metaphysics (형이상학)
- What is real? / 실재란 무엇인가요?
- What is the nature of consciousness? / 의식의 본질은 무엇인가요?
- Do we have free will? / 자유의지가 있나요?

### Ethics (윤리학)
- What makes an action right or wrong? / 행동의 옳고 그름을 결정하는 것은?
- What do we owe to others? / 우리는 타인에게 무엇을 빚지고 있나요?
- Can the ends justify the means? / 목적이 수단을 정당화할 수 있나요?

### Aesthetics (미학)
- What is beauty? / 아름다움이란 무엇인가요?
- What makes art meaningful? / 예술을 의미 있게 만드는 것은?
- Is taste objective or subjective? / 취향은 객관적인가요, 주관적인가요?

### Existential (실존적)
- What gives life meaning? / 삶에 의미를 부여하는 것은?
- How should we face mortality? / 죽음을 어떻게 마주해야 하나요?
- What does it mean to live authentically? / 진정성 있게 산다는 것은?

## Checklist

- [ ] Topic established clearly / 주제 명확히 설정
- [ ] Opening phase completed / 시작 단계 완료
- [ ] Used multiple question types (≥5 of 9) / 다양한 질문 유형 사용 (9개 중 5개 이상)
- [ ] Deep exploration achieved / 깊은 탐구 달성
- [ ] Synthesis phase completed / 종합 단계 완료
- [ ] Key insights identified / 핵심 인사이트 도출
- [ ] Open questions acknowledged / 열린 질문 인정
- [ ] Blog format generated / 블로그 형식 생성
- [ ] Language consistent throughout / 언어 일관성 유지
