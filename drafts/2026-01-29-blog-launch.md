## 왜 이 블로그를 시작했나

매일 AI와 대화하면서 많은 생각들이 오간다. 코딩 문제를 해결하기도 하고, 철학적인 주제로 깊은 대화를 나누기도 한다. 그런데 이 대화들이 휘발되는 게 아까웠다.

그래서 결심했다:
- AI와 나눈 대화를 기록하고
- 읽은 것들에 대한 생각을 정리하고
- 진행 중인 프로젝트를 문서화하자

## 오늘 만든 것

### 5개의 콘텐츠 제작 스킬

OpenCode에서 바로 사용할 수 있는 스킬들을 만들었다:

| 스킬 | 용도 |
|------|------|
| `/session-to-blog` | 코딩 세션을 블로그 포스트로 |
| `/philosophical-interview` | 소크라틱 대화 진행 |
| `/reading-digest` | 읽은 자료 정리 |
| `/project-journal` | 프로젝트 업데이트 |
| `/thought-dialogue` | 생각 탐구 대화 |

### Hugo 블로그 시스템

Mac Mini에서 24시간 운영할 수 있는 가벼운 블로그 시스템:
- Hugo + PaperMod 테마
- 한국어/영어 양쪽 지원
- 마크다운 기반 자동화

## 앞으로의 계획

1. **매일 기록하기**: 최소 하나의 대화나 생각을 기록
2. **철학적 인터뷰**: 주 1회 깊은 주제로 AI와 대화
3. **읽은 것 정리**: 책이나 기사를 읽으면 바로 정리
4. **프로젝트 업데이트**: 진행 중인 작업들 주간 정리

## 기술 스택

```
Hugo v0.155.0
Theme: PaperMod
Hosting: Mac Mini (예정)
Content: Markdown
Automation: OpenCode Skills
```

## 마치며

AI와 대화하는 것이 일상이 된 시대다. 그 대화들을 기록하고 공유하는 것이 의미 있다고 생각한다. 

이 블로그가 나만의 생각 정리 도구이자, 비슷한 관심사를 가진 사람들과 소통하는 창구가 되길 바란다.
